== Description ==

Advanced Notifications allows you to create beautiful custom notifications that appear on pages or posts of your choice.

= Features: =
Hereâ€™s a list of the Basic version features in plain text format:

**Notifications**

* Number of Notifications: Unlimited
* Types: Editor, Image, HTML
* Notification by User Role Types: Always all users & visitors, All users & visitors
* Scheduled Notification: No
* Notification by Devices Types: All devices, Desktop, Desktop & Tablet, Tablet, Only Tablet & Mobile, Mobile
* Delay Before Show: Yes
* Show Time: Yes
* Publish Type: All site pages, Custom post types & taxonomies, Only on specific pages
* Limiting the Number of Shows per Period: Yes
* Animations In & Out: Basic
* APIs for Developers: Yes

---

**Locations**

* Number of Locations: Unlimited
* Locations: Bottom & Left, Bottom & Right, Top & Left, Top & Right
* Custom Location Options: Basic
* Responsive: Yes

---

**Designs**

* Number of Designs: Unlimited
* Designs: Default, Success, Info, Warning
* Custom Design Options: Basic
* Responsive: Yes

---

**Triggers**

* Number of Triggers: Unlimited
* Number of Triggers per Notification: Only one
* Notifications Mode: Page is loaded and in triggers
* Triggers: Mouse leave from top, On click, On hover
* Trigger Action: Only show notification
